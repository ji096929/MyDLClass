## 📖 课程简介

本仓库为**东北大学悉尼智能科技学院**深度学习课程（专业选修课）的配套代码库。

课程从神经网络基础到多模态情感计算前沿（如有时间扩展），帮助学生建立扎实的理论基础与一定的实践能力。

---


## 📋 课程信息

| | |
|:--|:--|
| 📌 **课程性质** | 专业选修课，面向计算机科学与人工智能方向 |
| 📚 **主要内容** | MLP、CNN、RNN、Attention、Transformer 等 DL 基础及应用案例 |
| 🎯 **学习目标** | 跑通简单模型、理解核心概念、撰写技术报告 |
| 👨‍🏫 **主讲教师** | 傅昌锂 副教授（东北大学悉尼智能科技学院）|
| 🔬 **研究方向** | 多模态情感计算 · 多模态交互智能体 · AI for Cognitive Health |

---

## 🗺️ 课程大纲

| 章节 | 内容 | 关键词 |
|:----:|:-----|:-------|
| Ch1 | 神经网络 | Perceptron, MLP |
| Ch2 | 反向传播 | Backprop, Gradient |
| Ch3 | 多输入输出 | Multi-IO |
| Ch4-5 | 激活与损失 | ReLU, CrossEntropy |
| Ch6 | 卷积网络 | CNN, Conv2D |
| Ch7 | 循环网络 | RNN, Sequence |
| Ch8 | 长短时记忆 | LSTM, GRU |
| Ch9 | 词嵌入 | Word2Vec, Embedding |
| Ch10 | 编码解码器 | Encoder-Decoder, Seq2Seq |
| Ch11-12 | Transformer | Attention, BERT |
| Ch13-14 | 大模型基础 | Decoder-only, Autoregressive |
